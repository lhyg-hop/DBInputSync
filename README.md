# DBInputSync Secure Remote

一个偏自用的双通道输入同步工具。

它保留了“手机浏览器 + 手机输入法/语音输入 -> 电脑当前光标位置实时上屏”的体验，同时提供两种连接方式：

1. 同一局域网下优先走局域网直连
2. 不同网络下走 Cloudflare Quick Tunnel

这能显著提升实际使用中的可用性，但仍然不是“任何网络都绝对可用”。如果 `trycloudflare` 被拦、同时手机和电脑也不在同一局域网，跨网入口仍然会失败。

## 特性

- 手机继续使用浏览器和系统输入法，不需要原生 App。
- 同时提供局域网二维码和跨网二维码。
- 同网优先 LAN，异网使用 Cloudflare Quick Tunnel。
- 启动后生成一次性二维码和配对码。
- 所有写操作都必须先配对，再拿会话令牌访问。
- `undo` 按会话隔离，不会串到别的设备。
- 支持文本发送、回车、退格、方向键、符号包裹、撤销。
- 本地控制面板支持重新生成二维码、刷新局域网地址、暂停远程输入、紧急断开和重连隧道。

## 运行要求

### 桌面端

- Windows
- Python 3.9+
- `cloudflared` 已安装并且可在命令行直接执行

### 手机端

- 任意现代浏览器
- 可使用系统输入法和语音输入

## 安装

```bash
pip install -r requirement.txt
```

安装 `cloudflared` 后，确认下面命令可执行：

```bash
cloudflared --version
```

## 启动

```bash
python main.py
```

启动后程序会：

1. 启动本地服务和局域网访问入口
2. 探测当前局域网 IP
3. 尝试启动 Cloudflare Quick Tunnel
4. 在本地控制面板同时显示两种二维码

本地控制面板地址：

```text
http://127.0.0.1:5000/control
```

## 使用方式

1. 在电脑上启动程序。
2. 打开本地控制面板。
3. 如果手机和电脑在同一 Wi‑Fi / 同一局域网，优先扫“局域网直连”二维码。
4. 如果手机和电脑不在同一网络，扫“跨网连接（Cloudflare Quick Tunnel）”二维码。
5. 在手机页面输入文字或使用语音转文字。
6. 内容会同步到电脑当前光标位置。

## 安全说明

- 二维码中的配对码默认 120 秒过期。
- 配对码只能使用一次。
- 同一轮二维码中的局域网入口和跨网入口共用同一个配对码，任一入口兑换成功后另一入口立即失效。
- 手机页面会换取签名会话令牌，之后所有写操作都必须带令牌。
- 控制面板只允许从本机 `localhost/127.0.0.1` 打开。
- 远程输入可一键暂停，必要时可紧急断开跨网隧道。

## 热替换规则

依然支持 `hot-rule.txt`：

```txt
男主 = 张无忌
\s+ = 
```

规则会在发送文本前应用。
